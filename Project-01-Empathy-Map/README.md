# 🏋️ Project 01 – Empathy Map

## 📖 Project Overview

This project focuses on understanding the needs, behaviors, and challenges of busy office professionals before designing a fitness application.

The UX research process includes identifying the target audience, creating a user persona, and developing an empathy map to understand users' goals, pain points, and motivations.

---

## 🎯 Objective

To conduct user research and gain a deeper understanding of the target users before beginning the UI design process.

---

## 📂 Deliverables

- ✅ Target Audience
- ✅ User Persona
- ✅ Empathy Map

---

## 🛠️ Tools Used

- Canva
- UX Research
- User Persona
- Empathy Mapping
